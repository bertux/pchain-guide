PCHAIN's Wallet
===============

| PCHAIN's Wallet is wallet for PCHAIN. You can download it from `here <https://github.com/pchain-org/wallet/releases>`_.
| There are two mode in PCHAIN's wallet, one is light mode, which can use to transfer PI, delegate, create child chain, delpoy and interact with smart contract. The other mode is local node mode, which have another two functions including bidding for validator and apply for candidator.
