PIWallet
===============

| PIWallet is wallet for PCHAIN. You can download it from `here <https://github.com/pchain-org/wallet/releases>`_.
| There are two mode in PIWallet, one is light mode, which can use to

- transfer PI
- delegate
- create child chain
- delpoy and interact with smart contract

The other mode is local node mode, which can do all things above and also can

- vote for validator
- apply for candidate
